# app-legal-pages

This is a simple static website for legal pages (Privacy Policy and Terms of Use) that can be hosted on GitHub Pages.

## Files

- `index.html`: Home page
- `privacy-policy.html`: Privacy Policy page
- `terms-of-use.html`: Terms of Use (EULA) page
- `style.css`: Basic styling

## How to Publish on GitHub Pages

1. Push this repository to GitHub.
2. Go to the repository settings.
3. Under "Pages", select the main branch as the source.
4. The site will be available at `https://[username].github.io/app-legal-pages/`

## Usage

Replace the placeholder text in `privacy-policy.html` and `terms-of-use.html` with your actual content.
